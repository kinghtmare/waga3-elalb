### Hardware Components
- **Microcontroller**: ESP32-WROOM-32 Development Board 38-Pin
- **Motor Drivers**: 2x L293D/L298N modules
- **Servo Motor**: MG996R 360 degrees (hammer mechanism)
- **DC Motors**: 4x geared motors (148-600 RPM)
- **Sensors**: 5-channel line tracking sensor module
- **Power**: 12V rechargeable battery system
- **Connectivity**: Custom PCB with JST connectors

### Mechanical Design
- **Base Structure**: Two-tiered wooden platform
- **Materials**: Wood base with acrylic components
- **Hammer Mechanism**: Servo-controlled impact system
- **Sensor Array**: Curved configuration for optimal detection

### Software Architecture
- **Dual Operation Modes**: Autonomous and Manual control
- **Control Interface**: PS4 controller via Bluetooth
- **Sensor Integration**: Multi-sensor fusion from 5 IR sensors
- **Motor Control**: PWM-based velocity management
